# Console Pet Simulator


A simple console-based pet simulator game. You can adopt pets, take care of them using Random items, and monitor their stats over time. If you neglect them, they may die!

## Features
- Adopt pets of different types
- Stats: Hunger, Sleep, Fun
- Use Randomize items to improve pet stats
- Real-time stat decay
- Async item usage
- Events trigger on pet death
